Current Projects

Project Shift

Project Shift is a recent passion project that me and a small group of people are working on. It is a rhythm game consisting of 6 lanes but only 4 hittable buttons. To hit notes on other lanes, you have to shift your "Bracket," which indicates which 4 lanes you're able to hit. I am the main project leader, main programmer, and composer.

Moving On (Steam Page)

Moving On is a senior capstone project that me and 5 other students worked on. For our capstone project, we were imposed with many design constraints, including:

  • The game must be playable offline, and the total download size must be less than 100mb.
  • The game must be completeable using only a single hand, with no auditory information, and with colorblind support.
  • Active play can always reach a natural resting point within two minutes without the need to actively pause the game.
  • The game must have support for two different languages.

I was the design lead, composer, sound designer, assistant programmer, and assistant artist.

Gyle's Grand Prix

Gyle's Grand Prix (GGP for short) is a Minecraft Bedrock Realm created by a small group of friends as a passion project. GGP is a server where up to 10 players can race against each other on a variety of maps.

Completed Projects

Rusa: Tail of Gazelle

Rusa: Tail of Gazelle is a visual novel made for the SuNoFes 2024 game jam. In this game jam, teams had 2 months to create a visual novel or story-based game. We used Ren'Py as our game engine. I was a programmer, composer, and sound designer.

Wayfeather

Wayfeather is a simple sidescroller platformer made for the ZenoGameJam 6. Teams were given 1 week to create a game based on the theme "Going Up". The game was made in Godot. was the main programmer, composer, sound designer, and assistant level designer. This was the first game jam I participated in, and our game placed 9th overall out of 143 submissions.

Retro-Active

Retro-Active is a first-person 3D platformer that utilizes 2D games to interact and manipulate the world. This game was made for a class project for learning Unity. Each team was given a prompt for their game, and my team was given the prompt "game within the game". At the end of class, each team was able to vote a game for specific awards, such as "Aesthetic Award" and "Best Trailer". Retro-Active won the student award of "Tech Award"! I was the lead level designer, environmental artist, and assistant programmer.

Blorange

Blorange is a puzzle game made for a class project. We had to create a game using the Phaser framework. I was the composer, sound designer, graphics artist, and lead level designer.
